# Evidence Portfolio

This folder contains supporting evidence for the MyUserJourney platform. Organise materials into the subdirectories below.

---

## Folder Structure

```
evidence/
  screenshots/       Platform UI screenshots (dashboard, AI features, admin panel, privacy settings)
  whitepaper/         Exported whitepaper and supporting technical documents (PDF format)
  architecture/      System architecture diagrams, database schema visuals, data flow charts
  demo-videos/       Screen recordings demonstrating platform functionality and AI features
```

---

### whitepaper/
- Export `whitepaper.md` as a professionally formatted PDF
- Include any supplementary technical documents

### architecture/
- Database schema diagram (27 tables)
- System architecture overview (frontend, backend, database, AI layer)
- Data flow diagram (event collection pipeline)
- Privacy architecture diagram (consent flow, anonymisation, data subject rights)

### demo-videos/
- Platform walkthrough (3-5 minutes)
- AI features demonstration (predictive analytics, UX auditor, marketing copilot)
- Privacy compliance demonstration (consent flow, right to erasure, data export)
- Admin CMS demonstration (page creation, file management, user management)
